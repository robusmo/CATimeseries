Troubleshooting

  • Graph does not display even though I selected one or more variables.
  • Map does not show a variable when I select it.
  • I selected a layer but the map is coloured pink.
  • Can I publish a graph in a personal publication?

Graph does not display even though I selected one or more variables.

Graph creation can take some time: so first of all, have patience!
Then check if the selected area, point, box or geographical area, is over sea while a land layer has been selected. Check all dates that are available, and correct selected dates to available ones. If the problem persists, contact the web-site administrator (by email).

Map does not show a variable when I select it.

Using the metadata link or the information mask from the configuration graph, check that the selected month and year correspond with an available date. Check if the issue is due to a network problem. If the problem persists, contact the web-site administrator (by email).

Top

I selected a layer but the map is coloured pink.

This issue arises when there is a network problem on the machine that stores the selected layer. Try again later. If the problem persists, contact the web-site administrator (by email).

Can I publish a graph in a personal publication?

Before publishing something, please read the layer copyright on the metadata web-page (click on on "Table of Contents" section), or contained in the downloaded Pdf document.
